== Description ==

Shared Vision is a 3rd-party service that helps Creators manage their affiliate links on their website. In order to use this plugin, you must have a Creator account, which you can register for at https://joinsharedvision.com. We have a free plan available, using this plugin does not require you to be a paying customer of Shared Vision. Once you created your first Shopping List on the Shared Vision platform, you can use this plugin to quickly embed the list onto your Wordpress site. To learn more about Shared Vision, please see the following pages:

- https://joinsharedvision.com
- https://joinsharedvision.com/terms-and-conditions
- https://joinsharedvision.com/privacy-policy

The Shared Vision Wordpress plugin connects to your Shared Vision account granting your Wordpress website direct access to your shopping lists! You can now see your most recently created shopping lists in a dropdown, it can be aded to any page/post on your site. Take the guess work out of adding your Shared Vision lists!

The simple interface lets you easily choose a list. Anywhere on your site, where there is a list your readers can add items to their cart, making it easier to shop your post and checkout through the retailer, all while making sure your affiliate ID is  tagged in the shopping cart.

You can easily add your shopping lists to your blog post, compatible with the most popular Wordpress editors:

- Elementor
- Divi
- WPBakery
- Gutenberg

Features:
 - Add a shopping list to a post with one click
 - View most recently created shopping lists
 - Shopping Lists are automatically updated, no need to edit your post

If you like our plugin and find it useful, please leave a good rating/review.


== Frequently Asked Questions ==

= How to I create a Shared Vision Account? =

Go to https://creator.joinsharedvision.com/register?ref=wordpress&plan=proYearly to signup for your free account and a 30day trial!

= How do I get my API Key? =

When logged in to your Shared Vision account, go to: Settings => Integrations to get your API key

= What happens to my old embedded lists? =

Nothing! Your old embedded lists will work just as they have before. This Plugin is the newest way you can add lists to your post with one click.

== Installation ==

This section describes how to install the Shared Vision plugin and configuration.

1) Install

	* Go to the WordPress Dashboard “Add New Plugin” section.
	* Search For “Shared Vision.
	* Install, then Activate it.
2) Configure

	* Go to the Settings-> SharedVision Page
	* Add your API Key
